# AWS Capstone Project

## Objective
This project demonstrates the deployment of a **highly available, scalable, resilient, and secure web application** on AWS.

**Goals:**
- High Availability
- Scalability
- Security
- Resilience

## AWS Services Used
- VPC (Virtual Private Cloud)
- EC2 (Elastic Compute Cloud)
- EFS (Elastic File System)
- Auto Scaling Groups
- ALB (Application Load Balancer)
- Route 53 (DNS Management)
- NAT Gateway

## Project Phases
1. Design
2. Implementation
3. Testing & Optimization
4. Documentation

## Deliverables
- AWS Infrastructure (VPC, Subnets, NAT Gateway, EC2, EFS)
- Load-balanced and auto-scaled web application
- Route53 DNS with SSL enabled
